Etsy Seller Chrome Extension

Generate 13 SEO tags and full listings from inside Etsy shop manager. Floating panel on listing editor pages.

Install (2 minutes)

  1. Download seller-tools.zip from GitHub
  2. Unzip → open folder chrome-extension/
  3. Chrome → chrome://extensions → Developer mode ON
  4. Click Load unpacked → select chrome-extension folder
  5. Open Etsy → Your Shop → Edit a listing → panel appears bottom-right

Features

  • 13 SEO Tags — scrapes your title, opens tag generator pre-filled
  • Full Listing — opens ListingLab with your product data
  • Paste Tags — copy from tool, one-click paste into Etsy tag fields
